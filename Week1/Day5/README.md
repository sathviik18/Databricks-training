## 📌 Summary

In Day 5, I expanded my understanding of SQL window functions with more advanced analytical patterns. I worked on moving averages, percentile-based grouping using NTILE, and extracting first and last transactions per customer using ROW_NUMBER().

I also learned how to combine CTEs with aggregation to compute structured results like total sales per employee. Additionally, I practiced real-world analytics such as department payroll calculations and employee contribution percentages.

These techniques are widely used in business reporting, dashboards, and SQL-based data analysis interviews, making them essential for real-world data work.
