## 📚 Concepts Covered Today

- ROW_NUMBER() for unique ranking
- RANK() for ranking with gaps
- DENSE_RANK() for continuous ranking
- PARTITION BY for group-wise calculations
- LAG() for previous row comparison
- LEAD() for next row comparison
- Window functions vs GROUP BY difference
- Running totals and cumulative calculations
- Time-series style analytics using ORDER BY


## 📌 Summary

In Day 4, I learned SQL window functions that help perform advanced analytics without collapsing data. I practiced ranking techniques, group-wise calculations using PARTITION BY, and time-series analysis using LAG and LEAD. I also understood how window functions differ from GROUP BY by preserving row-level data while performing calculations.

These concepts are essential for real-world data analysis, reporting dashboards, and SQL interviews.
