---

## 🔹 1. REGEX Fundamentals

- Understanding pattern matching
- Using REGEXP_SUBSTR()

- Extracting structured text from strings

---

## 🔹 2. Character Matching Patterns

- Extracting numeric values
- Extracting alphabetic sequences

- Matching continuous character patterns

- `^` → Start of string
- `$` → End of string
- Extracting starting and ending patterns
- Position-based matching

- `+` → One or more matches
- `{2}` → Exact number of matches
- Continuous sequence extraction
- Controlled pattern matching

- Parsing structured email data

- Using lookbehind expressions

- Pattern stopping behavior
